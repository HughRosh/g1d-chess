# G1-D C++ Low-Level Control

This folder is for native Unitree SDK2 C++ code.

It uses:

- `rt/lowcmd`
- `rt/lowstate`
- `rt/secondary_imu`
- `LowCmd_`
- `LowState_`
- `IMUState_`

The Python framework should call higher-level wrappers later, but this folder keeps the official low-level DDS control path available.
